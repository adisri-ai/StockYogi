# StockYogi
A full-stack one-stop platform for all things stock market where users can view latest market updates along with live stock prices and track their portfolio. It also has additional features for AI-assistance and mutual fund investments.
# Project architecture
**Frontend** : ReactJS  
**Backend**  : NodeJS  
**Database** : MongoDB  
**LLM API**  : Mistral  
Additional Python backend for using python libraries such as yfinance for easy retrival of data.
# Functions Of Project
Access to live stock prices. 
Access to live market important updates
User can track their portfolio and add/remove stock as well
LLM-API based chatbot assistance for evaluating market sentiments.
